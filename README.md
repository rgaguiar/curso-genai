# IA Generativa Aplicada a Negócios

**Autor:** Rafael Aguiar  
**Formato:** 26 sábados · 4 horas por aula · 6 meses  
**Público-alvo:** Profissionais de negócios (RH, Jurídico, Marketing, Dados, Seguros)  
**Ambiente:** Google Colab + Agno Framework + Groq + OpenAI  

---

## O que é este curso?

Este curso transforma profissionais de negócio em construtores de agentes de IA. Não forma programadores — forma pessoas que usam IA como ferramenta de trabalho real, dentro do contexto da sua área.

Em 26 sábados você sai de zero para um portfólio público no GitHub com projetos funcionando no seu setor.

## Como usar este repositório

### Durante a aula
1. Abra o arquivo `.md` da aula no GitHub — está formatado para leitura direta
2. Copie cada bloco de código para o seu Google Colab na ordem apresentada
3. Rode célula por célula — não cole tudo de uma vez

### Para os exercícios extraclasse
- Cada módulo tem um arquivo em `exercicios/modN_exercicios.md`
- Use o notebook da aula como base — só troque `instructions`, `name` e `role`
- O resultado deve ser algo que você possa mostrar para um colega ou gestor

### Para o projeto final
- A partir do Módulo 6, você monta seu projeto no seu próprio repositório
- No final do curso você terá um repositório público com projetos funcionando
